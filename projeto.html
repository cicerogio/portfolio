Visão Geral

O FinFlow é um aplicativo de gestão financeira pessoal que estava enfrentando alta taxa de abandono e baixo engajamento dos usuários.

O FinFlow havia crescido rapidamente nos últimos dois anos, mas começou a enfrentar sérios problemas de retenção de usuários. A taxa de abandono após o primeiro uso era de 68%, e os usuários que permaneciam relatavam dificuldades para completar tarefas básicas como categorizar gastos e definir orçamentos.

Fui contratado para liderar uma iniciativa de redesign completo da experiência mobile, com foco em simplificar a jornada do usuário e aumentar o engajamento através de um design mais intuitivo e centrado nas necessidades reais dos usuários.

O projeto envolveu pesquisa extensiva com usuários, análise de dados comportamentais, prototipagem iterativa e testes de usabilidade para garantir que as soluções propostas realmente resolvessem os problemas identificados.

O Desafio

Alta Taxa de Abandono

68% dos usuários abandonavam o app após a primeira sessão, indicando problemas sérios no onboarding e primeiras impressões.

Interface Complexa

Usuários relatavam que as funcionalidades eram difíceis de encontrar e usar, com muitos passos desnecessários.

Baixo Engajamento

Usuários ativos usavam apenas 20% das funcionalidades disponíveis, sugerindo problemas de descoberta e utilidade.

Inconsistência Visual

Falta de um sistema de design coerente resultava em experiência fragmentada e confusa.

Metodologia

Utilizei uma abordagem de Design Thinking combinada com metodologias ágeis para garantir iterações rápidas e validação constante.

A Solução

Redesenhei completamente a arquitetura de informação e interface, focando em simplicidade e eficiência.

A solução final envolveu uma reestruturação completa da arquitetura de informação, simplificando a navegação e reduzindo o número de telas necessárias para completar tarefas essenciais.

Principais melhorias implementadas:

  • Onboarding reimaginado: Reduzi de 8 para 3 telas, focando no valor imediato para o usuário
  • Dashboard inteligente: Informações mais relevantes apresentadas de forma visual e acionável
  • Categorização automática: IA para categorizar gastos automaticamente, reduzindo trabalho manual
  • Navegação simplificada: De 5 abas principais para 3, com funcionalidades reagrupadas logicamente
  • Design system unificado: Componentes consistentes em toda a aplicação

O design final priorizou clareza visual, redução de carga cognitiva e fluxos mais diretos para as tarefas mais comuns dos usuários.

Além das métricas quantitativas, recebemos feedback muito positivo dos usuários sobre a nova experiência. A equipe de suporte reportou uma redução de 40% nas consultas relacionadas a dificuldades de uso.

O projeto também estabeleceu uma base sólida para futuras iterações, com o sistema de design criado sendo usado para acelerar o desenvolvimento de novas funcionalidades.
